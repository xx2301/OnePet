# OnePet - Pet Simulation Game on One Blockchain

## Overview
OnePet is a comprehensive pet simulation game built on the One Blockchain. 
Players can adopt, raise, and battle with virtual pets while earning rewards.

## Quick Start

### Smart Contracts
```bash
# Build contracts
one move build

# Run tests  
one move test

# Deploy (after build)
one client publish --gas-budget 100000000
```
